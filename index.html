highlighter.js

Highlight some body text! This is a jQuery plugin that allows you to display a pop-up tooltip when a user highlights text. This can be useful in a variety of situations, and there didn't seem to be a good open source solution before this. See the github README for details on how to implement this on your website. It's very easy!

Feel free to use the github page to open any issues or submit pull requests for any fixes. We are very open. This was created by @mpconlen for HuffPostLabs
